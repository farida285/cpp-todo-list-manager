# C++ To-Do List Manager

A simple console-based To-Do List Manager built using C++.  
This project allows users to manage their daily tasks through a menu-driven interface.

The program demonstrates the use of core C++ concepts such as:
- Structures
- Enumerations
- Functions
- Input validation
- Array manipulation
- Program organization using modular functions

---

## Features

The application supports the following operations:

1. Add a new task
2. View all tasks
3. View pending tasks
4. View completed tasks
5. View task details
6. Edit a task (title, description, or priority)
7. Mark a task as done
8. Mark a task as pending
9. Delete a task
10. Delete all tasks
11. Exit the program

---

## Task Information

Each task contains:

- Task Number
- Title
- Description
- Priority (High / Medium / Low)
- Status (Pending / Done)

---

## Input Validation

The program includes input validation for:

- Menu selection
- Task numbers
- Yes/No confirmations
- Title and description length

- Basic data management logic

## How to Run

1. Compile the program:
 g++ todo-list-manager.cpp -o todo
2. Run the executable:
 ./todo
